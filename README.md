# 歌词旋律配对小程序

一个专为独立音乐人设计的歌词与旋律配对工具，帮助你轻松管理创作、录制旋律并实现完美配对。

## 功能特色

### ✨ 核心功能
- 📝 **歌词管理**: 创建、编辑、保存多段歌词
- 🎵 **旋律录制**: 支持录制或上传音频文件
- 🔗 **配对关联**: 将歌词与对应旋律精准关联
- ▶️ **智能播放**: 试听旋律时自动同步显示歌词
- 📂 **作品库**: 统一管理所有创作作品
- 📋 **歌词模板**: 6种预设模板，快速开始创作
- ⏱️ **节拍器**: 专业节拍器，保持精准节奏
- 📊 **统计面板**: 创作数据分析与可视化
- 🌙 **深色模式**: 护眼的夜间主题
- 📤 **导出工具**: 支持TXT、JSON导出和分享

### 🎯 特色功能
- 段落类型标记（主歌、副歌、桥段等）
- 创作灵感笔记记录
- 多维度搜索筛选
- 作品分类标签管理
- 优雅的渐变色UI设计
- 云端数据同步

## 技术栈

- **前端**: 微信小程序原生开发
- **数据库**: 微信云开发数据库
- **存储**: 微信云存储
- **音频**: 微信小程序录音与播放API

## 快速开始

### 1. 环境准备
- 安装微信开发者工具
- 注册微信小程序账号
- 开通云开发功能

### 2. 配置项目
1. 打开微信开发者工具，导入项目
2. 在 `app.js` 中替换 `env: 'your-env-id'` 为你的云环境ID
3. 在 `project.config.json` 中替换 `appid: 'your-appid'` 为你的小程序AppID

### 3. 初始化云开发
1. 在开发者工具中点击「云开发」按钮
2. 创建数据库集合 `works`
3. 创建云存储文件夹 `melodies`
4. 设置相应的读写权限

### 4. 运行项目
1. 点击「编译」按钮
2. 在模拟器中预览效果
3. 可以在真机上测试录音功能

## 使用指南

### 创作新作品
1. 点击底部「创作」标签或首页「创建新作品」
2. 填写作品标题、主题和标签
3. 添加多段歌词，选择段落类型
4. 为每段歌词录制或上传旋律
5. 添加创作灵感笔记
6. 点击「保存作品」

### 管理作品
1. 在「作品库」中查看所有作品
2. 使用搜索框查找特定作品
3. 使用筛选标签快速定位
4. 编辑或删除已有作品

### 试听播放
1. 点击作品进入播放器页面
2. 点击「播放」按钮试听旋律
3. 歌词会自动滚动到当前播放段落
4. 支持自动播放下一段旋律
