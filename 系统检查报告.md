## 📋 GitHub CLI 安装指南

### 方法1：等待网络恢复后使用 Homebrew（推荐）

当网络恢复正常后：
```bash
brew install gh
```

### 方法2：手动下载安装（当前可用）

1. **访问 GitHub CLI 下载页面**：
   在浏览器中打开：https://github.com/cli/cli/releases

2. **下载 macOS 版本**：
   - 找到最新版本（当前是 v2.88.1）
   - 下载：`gh_2.88.1_macOS_amd64.tar.gz`

3. **解压并安装**：
   ```bash
   # 解压到临时目录
   tar -xzf ~/Downloads/gh_2.88.1_macOS_amd64.tar.gz -C /tmp

   # 复制到 /usr/local/bin
   sudo cp /tmp/gh_2.88.1_macOS_amd64/bin/gh /usr/local/bin/

   # 赋予执行权限
   sudo chmod +x /usr/local/bin/gh

   # 验证安装
   gh --version
   ```
