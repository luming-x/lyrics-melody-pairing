### 可选：配置云开发（要使用完整功能）

如果需要保存作品、录音等功能，需要配置云开发：

#### 步骤1：开通云开发
1. 在开发者工具中点击「云开发」按钮
2. 按提示开通（免费版即可）
3. 复制云环境ID

#### 步骤2：配置环境ID
打开 `app.js` 文件，找到第20行左右：
```javascript
globalData: {
  userInfo: null,
  envId: null // 云环境ID，配置后填写
}
```
将 `null` 改为你的云环境ID，例如：
```javascript
envId: 'your-cloud-env-id'
```

#### 步骤3：创建数据库
1. 点击「云开发」→「数据库」
2. 创建集合：`works`
3. 权限设置为「所有用户可读写」

#### 步骤4：创建存储文件夹
1. 点击「云开发」→「存储」
2. 创建文件夹：`melodies`
