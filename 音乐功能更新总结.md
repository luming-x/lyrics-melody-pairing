# 音乐功能更新总结

## 🎵 更新概述

本次更新为歌词旋律配对工具新增了完整的音乐创作辅助功能，包括简谱生成、虚拟乐器、和弦库、音高识别等专业音乐工具，极大提升了创作效率和专业性。

## 📋 新增功能列表

### 1. 📝 简谱自动生成功能
**文件位置**: `pages/sheet-music/`, `utils/sheet-music-generator.js`

**功能特性**:
- 自动将旋律转换为简谱格式
- 支持12种调式（C、C#、D、D#、E、F、F#、G、G#、A、A#、B）
- 支持4种拍号（2/4、3/4、4/4、6/8）
- 可选择是否显示歌词
- 支持复制简谱文本
- 支持导出为图片保存
- 优雅的渐变UI设计

**核心算法**:
- 频率到音高的精确转换（基于A4=440Hz标准）
- 支持升降音处理（#、b）
- 调式转调功能
- 音符时长映射（全音符、二分音符、四分音符等）

**使用场景**:
- 将录制的旋律快速转换为简谱
- 为创作作品生成专业乐谱
- 导出分享给其他音乐人

---

### 2. 🎹 钢琴虚拟键盘组件
**文件位置**: `components/piano-keyboard/`, `pages/piano/`

**功能特性**:
- 可配置八度范围（3-5八度）
- 标准黑白键布局
- 触摸交互支持
- 实时显示当前音符
- 演奏历史记录（最近10个音符）
- 八度范围自定义设置
- 音符频率精确计算
- 振动反馈增强体验

**技术实现**:
- 完整的组件化设计
- 基于频率的音高计算
- 自定义事件系统（noteon/noteoff）
- 响应式布局适配
- 支持外部API调用

**使用场景**:
- 创作时试听和弦进行
- 练习音阶和琶音
- 配合节拍器进行节奏练习

---

### 3. 🎸 吉他和弦库功能
**文件位置**: `pages/chords/`, `utils/chord-data.js`

**功能特性**:
- 28个常用和弦（C、D、E、F、G、A、B调）
- 4种和弦类型（大三和弦、小三和弦、属七和弦、大七和弦）
- 详细的和弦指法图表
- 按调式分类浏览
- 按和弦类型筛选
- 和弦搜索功能
- 转调支持（半音转调）

**和弦数据**:
- 每个和弦包含完整的品位数据
- 标准指法说明
- 可视化琴颈图表
- 6根弦的详细配置

**使用场景**:
- 快速查找和弦指法
- 学习不同调式的和弦
- 为创作选择合适的和弦进行

---

### 4. 🎤 音高识别功能
**文件位置**: `pages/pitch-recognition/`, `utils/pitch-detector.js`

**功能特性**:
- 实时音高检测（基于自相关算法）
- 精确的频率到音高转换
- 音准偏差显示（音分精度）
- 调性自动分析
- 调性置信度评估
- 音高历史记录
- 支持音频文件处理

**核心算法**:
- 自相关音高检测算法
- 抛物线插值提高精度
- 十二平均律音高映射
- 调式分析算法
- 和弦检测算法

**技术特点**:
- 44.1kHz采样率
- 单声道音频处理
- 实时帧处理
- 振动反馈提示

**使用场景**:
- 演唱音准练习
- 乐器调音
- 录音质量检查
- 音高训练

---

### 5. 🎼 音调分析功能
**文件位置**: 集成在 `utils/pitch-detector.js` 中

**功能特性**:
- 自动分析音乐调性
- 支持12个大调调式检测
- 调性置信度评估
- 音阶分析
- 主音和属音识别

**分析原理**:
- 统计音符出现频率
- 匹配调式音阶模式
- 权重计算（主音、属音）
- 置信度算法

**使用场景**:
- 分析作品的调性
- 辅助和弦选择
- 音乐理论学习

---

### 6. 🎵 和弦检测功能
**文件位置**: 集成在 `utils/pitch-detector.js` 中

**功能特性**:
- 自动识别和弦类型
- 支持三和弦识别
- 大三和弦、小三和弦检测
- 减三和弦、增三和弦检测
- 根音识别

**检测类型**:
- Major（大三和弦）
- Minor（小三和弦）
- Diminished（减三和弦）
- Augmented（增三和弦）

**使用场景**:
- 分析录音中的和弦
- 学习和弦识别
- 即兴伴奏辅助

## 📚 功能使用指南

### 简谱生成
1. 进入"作品库"或"创作"页面
2. 选择作品进入简谱生成页面
3. 调整调式和拍号
4. 点击"保存图片"导出

### 虚拟钢琴
1. 从首页进入"虚拟钢琴"
2. 点击琴键演奏
3. 查看演奏历史
4. 调整八度范围设置

### 吉他和弦库
1. 进入"和弦库"
2. 选择调式或类型筛选
3. 点击和弦查看指法
4. 查看指法说明

### 音高识别
1. 进入"音高识别"
2. 点击"开始识别"
3. 演唱或演奏
4. 查看实时音高和调性分析
